⚙ Configuration requise — Apps Script non connecté

  1. Ouvrez votre Google Sheet CEBS / CEJ / AEBB
  2. Menu Extensions › Apps Script
  3. Collez le code du fichier DRRO_AppsScript.gs fourni
  4. Remplacez API_TOKEN par votre token secret (ex: DRRO_SECRET_2026)
  5. Cliquez Déployer › Nouvelle application Web → accès "Tout le monde"
  6. Copiez l'URL → collez-la dans ZONE_CONFIG.CEBS.apiUrl du fichier HTML
  7. Faites de même pour CEJ et AEBB
